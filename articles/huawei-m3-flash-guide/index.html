华为平板 M3 扩容与刷机指南

DEVICE: BTV-DL09 (64GB) // KIRIN 950

本教程基于实际操作记录，针对华为平板 M3 (BTV-DL09 64GB 版)。严格按步骤操作即可完成扩容和刷机。

⚠️ 风险提示

  • 数据会全部清除：请提前备份到电脑或 SD 卡
  • 分区操作有变砖风险：命令务必复制粘贴，不要手动输入

准备工作

必需条件

  1. 已解锁 Bootloader：Fastboot 界面显示 PHONE Unlocked
  2. ADB 工具：必须使用 v23.0.1 版本，新版会导致 partition size: 0 假报错
  3. TWRP 镜像：适配当前系统的 twrp.img
  4. 底层工具包：parted、mkfs.ext4、e2fsck、resize2fs
  5. ROM 和 GApps：放入 MicroSD 卡中
